Graph Architecture & State Schema

The LangGraph orchestrator implements an 11-node state machine with strict type checking, resilient conditional routing, and state transition checkpointers.

State Reducers Mapping

Field Reducer Type Why
messages Append (list[str]) Audit logs / Conversation history
route Overwrite (str) Current path decision
evaluation_result Overwrite (str) Retry loop evaluation
approval Overwrite (dict) HITL reviewer and decision metadata
tool_results Append (list[str]) Preserves history of tool logs
events Append (list[dict]) Standardized audit events tracking

Robust Self-Recovery

1. Transient Tool Failures: Evaluator node dynamically captures error streams from downstream nodes, routing the state back into the retry loop. Attempt bounds protect against endless loops.

2. Risky Operations: Operations classification like refunds and deletions are routed via an interrupt layer. If LANGGRAPH_INTERRUPT=true, it triggers a checkpointer suspend state.
